# ATM-Banking-System
This project is developed using core C programming concepts including conditional statements, loops, switch-case, and standard input/output operations. It is designed to help beginners understand real-world banking logic, user interaction, and simple security mechanisms in software systems.

The project is suitable for college mini-projects, lab assignments, and resume building, and serves as a strong foundation for learning more advanced concepts like file handling and database integration.

## Technologies Used

Language: C

Compiler: CodeBlocks

Platform: Console-based (Windows)

## Features

 - PIN-based authentication

- Balance inquiry

- Cash withdrawal with validation

- Cash deposit

- Menu-driven interface

- Secure exit option

## Future Enhancements

- File handling for storing user data

- Multiple account support

- Transaction history

- PIN change functionality
